控制器 Controller Link Layer + PHY

核心层详解

PHY (Physical Layer)

  • RF射频收发
  • GFSK调制解调
  • 发射接收切换
  • Channel Index → RF频率

Link Layer

  • 状态机: Standby/Adv/Scann/Initiator/Connected
  • 广播/扫描/连接
  • 数据加解密
  • CRC校验

HCI (Host Controller Interface)

  • Command/Event/ACL/SCO
  • Flow Control
  • Buffer Management

L2CAP

  • Channel ID (CID) 复用
  • Segmentation/Reassembly
  • Flow Control
  • LE Credit Based Flow

HCI指令类型

类型 OGF 示例 说明
Link Control 0x01 HCI_Inquiry 扫描设备
Link Policy 0x02 HCI_Accept_Connection 连接管理
Host Controller 0x03 HCI_Set_Event_Mask 控制器配置
Information 0x04 HCI_Read_Local_Version 版本信息
LE Controller 0x08 HCI_LE_Create_Connection BLE操作
