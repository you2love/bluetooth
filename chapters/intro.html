蓝牙核心概念

蓝牙是一种工作在 2.4GHz ISM 频段的短距离无线通信协议，采用跳频扩频 (FHSS) 技术实现抗干扰，支持经典蓝牙 (BR/EDR) 和低功耗蓝牙 (BLE) 两种模式。

技术规格详解

📡 频段与信道

2.4GHz ISM 频段 (2400-2483.5 MHz)

  • BLE: 40 个信道 (每 2MHz)
  • 经典蓝牙：79 个信道 (每 1MHz)
  • 跳频：1600hop/s (BLE) / 1600hop/s (经典)

📊 调制方式

GFSK (高斯频移键控)

  • BT=0.5, 调制指数=0.45-0.55
  • BLE 1M PHY: 1Mbps
  • BLE 2M PHY: 2Mbps
  • BLE Coded PHY: 500kbps/125kbps

🔋 功率等级

  • Class 1: 100mW (+20dBm), 100m
  • Class 2: 2.5mW (+4dBm), 10m
  • Class 3: 1mW (0dBm), 1m

🔒 调制差异

  • 经典蓝牙：BR/EDR, EDR(π/4-DQPSK, 8DPSK)
  • BLE: 单模设备，无法与经典蓝牙兼容

双模 vs 单模

类型 支持模式 典型应用 芯片
双模 (BR/EDR+BLE) A2DP, HFP, SPP + GATT 手机、平板、PC Qualcomm, Broadcom
单模 (BLE Only) GATT only IoT 设备、传感器 Nordic, TI, Dialog
