蓝牙核心概念

蓝牙是一种工作在2.4GHz ISM频段的短距离无线通信协议，采用跳频扩频(FHSS)技术实现抗干扰，支持经典蓝牙(BR/EDR)和低功耗蓝牙(BLE)两种模式。

技术规格详解

📡 频段与信道

2.4GHz ISM频段 (2400-2483.5 MHz)

  • BLE: 40个信道 (每2MHz)
  • 经典蓝牙: 79个信道 (每1MHz)
  • 跳频: 1600hop/s

📊 调制方式

GFSK (高斯频移键控)

  • BLE 1M PHY: 1Mbps
  • BLE 2M PHY: 2Mbps
  • BLE Coded: 500/125kbps

🔋 功率等级

  • Class 1: 100mW, 100m
  • Class 2: 2.5mW, 10m
  • Class 3: 1mW, 1m

🔒 模式差异

  • 经典蓝牙: 音频/文件
  • BLE: IoT/可穿戴
