蓝牙核心概念

蓝牙是一种工作在2.4GHz ISM频段的短距离无线通信协议，采用跳频扩频(FHSS)技术实现抗干扰。

💻 代码示例: 读取电池电量

// Android (Java/Kotlin)
BluetoothManager manager = getSystemService(BluetoothManager.class);
BluetoothGatt gatt = device.connectGatt(this, false, gattCallback);

// 1. 发现服务
gatt.discoverServices();

// 2. 在Callback中获取电池服务
BluetoothGattService batteryService = gatt.getService(UUID.fromString("180f"));
BluetoothGattCharacteristic batteryLevel = batteryService.getCharacteristic(UUID.fromString("2a19"));

// 3. 读取数据
gatt.readCharacteristic(batteryLevel);

// 4. 订阅通知
gatt.setCharacteristicNotification(batteryLevel, true);

💻 代码示例: 写入控制命令

// 写入数据控制LED
byte[] data = new byte[] { 0x01 }; // 开灯

BluetoothGattCharacteristic ledChar = ledService.getCharacteristic(
    UUID.fromString("0000ff01-0000-1000-8000-00805f9b34fb"));

ledChar.setValue(data);
ledChar.setWriteType(BluetoothGattCharacteristic.WRITE_TYPE_DEFAULT);
gatt.writeCharacteristic(ledChar);
